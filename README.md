## 🎯 Problem Statement

Cab services generate massive data, but without analytics:
- No visibility into cancellations  
- No revenue tracking  
- No customer behavior insights  

👉 This project solves these challenges using an **end-to-end data analysis pipeline**

---

## ⚙️ Tech Stack

|----------|--------|
| Excel     | Data Cleaning |
| MySQL     | Data Storage & Queries |
| Python    | EDA (Pandas,MySQL Connector) |
| Power BI  | Dashboard & Visualization |
